## Get started in three steps

Get your documentation site running locally and make your first customization.

### Step 1: Getting Access Code

<AccordionGroup>
  <Accordion title="Where to find a code?" icon="key">
    You can get an access code inside our Discord. [Join our server](disc), ask a code in the general chat, and members will pass you one.
  </Accordion>
  <Accordion title="How to use my code?" icon="unlock-keyhole">
    2. Start the [telegram bot](https://t.me/@SorcererTrading_bot).
    3. Enter your code in the chat.

    <Tip>
      You’ll unlock full access to all features if the code is valid.
    </Tip>
  </Accordion>
</AccordionGroup>

### Step 2: Activating the Web Extension

<AccordionGroup>
  <Accordion title="Download Sorcerer Extension" icon="arrow-down-to-line">
    Download the Sorcerer extension from the [Chrome Web Store](https://dashboard.mintlify.com/settings/organization/github-app).
  </Accordion>
  <Accordion title="Link your Telegram" icon="plug">
    To enable it in your browser you must:

    1. Open [Sorcerer on Telegram](https://t.me/@SorcererTrading_bot).
    2. Tap the  `"Sorcerer Web Extension"` button.
    3. Copy your unique access code and paste it into the extension.
  </Accordion>
</AccordionGroup>

### Step 3: Your Referral Code

<Accordion title="Set your referral code" icon="comment-code">
  To get a referral code:

  1. Open [Sorcerer on Telegram](https://t.me/@SorcererTrading_bot).
  2. Tap the  `"Referrals"` button.
  2. Tap the  `"Change Ref Code"` button.
  3. Enter a unique code in the chat.
</Accordion>

<Accordion title="Sharing you Code" icon="share-nodes">
  Feel free to share your code anywhere. You earn up to 33% instantly from your referrals’ fees.
</Accordion>
